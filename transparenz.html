Designprinzipien dieser Website

Open Source

Diese Website und der Programm-Code wurde von mir eigenständig erstellt. Mein altes Entwickler-Herz versucht immer wieder zu verstehen, wie etwas funktioniert. Ausprobieren, Fehler machen, lernen und verzweifelt nach dem Schreibfehler im Code suchen. Der Bauplan und Source Code dieser Seiten kann von allen kopiert, überprüft, modifiziert und verbessert werden. Er ist komplett auf Github veröffentlich. Dort finden Sie eine ausfürliche Dokumentation dieser Website. Ich bin sehr dankbar für Verbesserungsvorschläge und Ideen für eine zukunftsorientiertere Gestaltung.

  • Mobile, Cloud und Security First Design Prinzipien
  • Einfach skalierbare und flexible Betriebsumgebung
  • Schnelles weltweit verteiltes Content Delivery Network (CDN)
  • Einfach zu lesende statische HTML und CSS Seiten
  • Keine Datenbank, keine Patches und reduzierte Angriffsszenarien
  • Kein Java Script oder spezielle Browsereinstellungen erforderlich
  • Schnelle und einfache Übertragbarkeit auf beliebige Umgebungen
  • Einfaches Aktualisieren auch von jedem Smartphone oder Tablet

Weitere Informationen zu den verwendeten digitalen Komponenten dieser Seiten sowie der Source Code kann hier unter Github abgerufen werden.

Server Log-Dateien

Wenn Sie meine Website aufrufen, ist es technisch notwendig, dass über Ihren Internetbrowser Daten an unseren Webserver übermittelt werden. Folgende Daten werden während einer laufenden Verbindung zur Kommunikation zwischen Ihrem Internetbrowser und dem externen Webserver aufgezeichnet:

  • Datum und Uhrzeit der Anforderung
  • Name der angeforderten Datei
  • Seite, die die Datei angefordert hat
  • Zugriffsstatus
  • Verwendeter Webbrowser/Betriebssystem
  • IP-Adresse des anfordernden Rechners
  • Übertragene Datenmenge
Die aufgelisteten Daten erhebe ich, um einen reibungslosen Verbindungsaufbau der Website zu gewährleisten und eine komfortable Nutzung meiner Website durch die Nutzer zu ermöglichen. Zudem dient die Logdatei der Auswertung der Systemsicherheit und -stabilität sowie administrativen Zwecken.
